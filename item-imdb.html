IMDb Top 250 Prediction

Make a prediction on whether a new movie would be in the top250 of IMDb

- Completed this academic project with knowledge of data mining by three procedures: gather existing data, analyze the data, and predict the training data

- Implemented a Python script to scrape user reviews from IMDb website: using Beautiful Soup module to look into the HTML component of the website, and then saving these reviews as txt files.

- Analyzed the txt files and decided which words represents for positive (good movie) and for negative (bad movie).

- Put the training data (a new movie) to predict whether it would be in the Top 250. The results proved that the accuracy was over 80%.
