Student Attendance Signup System

Optimize the process for teachers/professors to take attendance

- Full Stack Development using Java, XML, PHP, JSON, MySQL, and HTTP request.

- Developed and deployed using Android Studio; used Git for version control.

- Reduced the time for taking attendance by 4 minutes. More people, more reduction.
